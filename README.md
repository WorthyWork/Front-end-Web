## Installation

npm Initialize:
```console
$ npm init
$ npm cd client && npm init
$ npm run init
```

Firestore Initialize:
```console
$ mkdir secret
```
get firebase-adminsdk.json and move to /secret dir

## Quick Start

### Start Front-end & Back-end Server in Development mode:
```console
$ npm run dev
```

### Auto-Geneterate Swagger Doc
```console
$ npm run swagger-autogen
```
