## Installation

Initialize front-end:
```console
$ cd client
$ npm install express
```

Initialize front-back:
```console
$ cd server
$ npm install express
```

## Quick Start

### Start Front-end Server:
```console
$ cd client
$ npm start
```

### Start Back-end Server:
#### Online Mode:
```console
$ cd server
$ npm start
```
#### Offline Mode:
```console
$ cd server
$ npm run dev
```

### Auto-Geneterate Swagger Doc
```console
$ cd server
$ npm run swagger-autogen
```
